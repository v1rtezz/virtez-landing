# Быстрый SEO-план для брендовых запросов

Цель: как можно быстрее добиться нахождения сайта по запросам:
- `Virtez VPN`
- `Виртез впн`

## Минимум, который нужно сделать сразу

1. Усилить бренд-сигнал на главной странице:
- В `title`, `h1` и `meta description` явно указать:
  - `Virtez VPN`
  - `Виртез ВПН`
- Добавить JSON-LD с названием бренда (Organization/WebSite).

2. Добавить базовую техническую SEO-обвязку:
- `canonical`
- `robots.txt`
- `sitemap.xml` (минимум с URL `https://virtez.pro/`)

3. Подключить и настроить вебмастера:
- Google Search Console
- Яндекс Вебмастер
- Bing Webmaster
- Отправить `sitemap.xml` во все кабинеты.

4. Запросить ускоренный переобход главной:
- Google: URL Inspection -> Request indexing
- Яндекс: Индексирование -> Переобход страниц

5. Обновления отправлять через IndexNow:
- После важных правок отправлять URL главной страницы.

6. Дать первые брендовые ссылки:
- Сделать 2-5 внешних упоминаний с прямой ссылкой на `https://virtez.pro/`:
  - Telegram-канал/посты
  - Профили
  - Тематические каталоги

## Ожидание по срокам
- Первые попадания по брендовым запросам могут появиться за несколько дней.
- В отдельных случаях индексация занимает от нескольких дней до нескольких недель.
